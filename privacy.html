DropZone — Privacy Policy

Last updated: July 2, 2026

DropZone is a file-transfer app that sends files from your Android device to a browser on another device using a QR code and an end-to-end relay connection. This page explains what data the app touches and what happens to it.

Camera

DropZone requests camera access solely to scan the QR code shown on the receiving browser. Barcode scanning runs entirely on your device using an on-device ML Kit model. No camera image, video, or scan result is ever uploaded, stored, or transmitted anywhere — it is used only to read the connection details encoded in the QR code (a session ID and server address).

Files you choose to send

DropZone only accesses files you explicitly select through the Android file picker. Selected files are transmitted, in encrypted chunks, directly to the session you scanned via QR code. Files are relayed in real time and are never written to disk or stored on the server — the server holds them in memory only for the duration of the transfer and discards them immediately after.

Session data

Each transfer uses a temporary session (an 8-character ID, file name, and file size) held in server memory. Sessions automatically expire and are deleted after 10 minutes, whether or not a transfer completed. There is no database and no persistent storage of session or file information.

What we don't do

  • No user accounts, sign-in, or persistent user identifiers
  • No advertising or ad SDKs
  • No analytics or tracking SDKs
  • No sale or sharing of data with third parties
  • No storage of transferred files beyond the live relay

Network & security

All communication between the app, the server, and the receiving browser is encrypted in transit (wss:// / TLS). As with any server, standard connection metadata (such as IP address and timestamps) may be visible to the server for operational and abuse-prevention purposes during an active connection, consistent with normal web server behavior.

Children's privacy

DropZone is not directed at children and does not knowingly collect data from children.

Changes to this policy
